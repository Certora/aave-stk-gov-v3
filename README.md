# Stake token rev 3

Repository contains a new revision of `StakedTokenV3` and `StakedAaveV3` with improved slashing mechanism and updated to solc 0.8.

## Development

### Install

```sh
cp .env.example .env
forge install
```

### Test

```sh
forge test
```

### Diff

Generate a code & storage layout diff.

```sh
npm run diff
```
